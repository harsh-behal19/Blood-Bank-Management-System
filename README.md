🩸 Blood Bank Management System

A command-line based Blood Bank Management System developed using Python and MySQL.
This project helps manage donor and receiver records efficiently through a structured menu-driven interface.

📌 Project Overview

The Blood Bank Management System is designed to maintain and organize blood donor and receiver information in a database. It allows users to perform complete record management operations while tracking blood stock based on blood groups.

The system runs smoothly on VS Code as well as Command Line / Terminal.

🛠 Technologies Used

Python 3

MySQL

mysql-connector-python

VS Code / Command Prompt

📊 Blood Stock

The system automatically calculates available blood stock by counting donors group-wise, helping track availability efficiently.
